> [!NOTE]
> To clarify, these only work with games which don't filter out **Unity Rich Text Tags**.
> There are also many ways to do the same thing, so I have listed all of them here.
---

| Type | Format | Other Formats |
|------|--------|----------------|
| AllCaps | `<allcaps>text</allcaps>` | `<allcaps>text` |
| Align | `<align=center>text</align>` | `<align=left>text</align>`, `<align=right>text</align>`, `<align=center>text`, `<align=left>text`, `<align=right>text`, `<align=justified>text</align>`, `<align=flush>text</align>` |
| Alpha | `<alpha=#AA>text</alpha>` | `<alpha=#AA>text` |
| Bold | `<b>text</b>` | `<b>text` |
| Color | `<color=#RRGGBB>text</color>` | `<#RRGGBB>text`, `<#RGB>text` |
| Font | `<font="FontName">text</font>` | `<font="FontName">text` |
| Font Weight | `<font-weight=700>text</font-weight>` | `<font-weight=700>text` |
| Gradient | `<gradient="PresetName">text</gradient>` | `<gradient="PresetName">text` |
| Highlight / Mark | `<mark=#RRGGBB>text</mark>` | `<mark=#RGB>text</mark>`, `<mark=#RRGGBB>text`, `<mark=#RGB>text` |
| Indent | `<indent=10>text</indent>` | `<indent=10>text` |
| Line Break | `<br>` | – |
| Line Height | `<line-height=1.5>text</line-height>` | `<line-height=1.5>text` |
| Line Indent | `<line-indent=10>text</line-indent>` | `<line-indent=10>text` |
| Link | `<link="id">text</link>` | `<link="id">text` |
| Lowercase | `<lowercase>text</lowercase>` | `<lowercase>text` |
| Margin | `<margin=10>text</margin>` | `<margin=10,10>text</margin>`, `<margin=10>text` |
| Monospace | `<mspace=1>text</mspace>` | `<mspace=1>text` |
| No Break | `<nobr>text</nobr>` | `<nobr>text` |
| No Parse | `<noparse><b>text</b></noparse>` | – |
| Page Break | `<page>` | – |
| Position | `<pos=100>text</pos>` | `<pos=100>text` |
| Quad | `<quad material=1 size=20 x=0.1 y=0.1 width=0.5 height=0.5>` | – |
| Rotate | `<rotate=45>text</rotate>` | `<rotate=45>text` |
| SmallCaps | `<smallcaps>text</smallcaps>` | `<smallcaps>text` |
| Space | `<space=10>text` | – |
| Sprite | `<sprite=0>` | `<sprite name="icon">` |
| Strikethrough | `<s>text</s>` | `<s>text` |
| Subscript | `<sub>text</sub>` | `<sub>text` |
| Superscript | `<sup>text</sup>` | `<sup>text` |
| Style | `<style="StyleName">text</style>` | `<style="StyleName">text` |
| Underline | `<u>text</u>` | `<u>text` |
| Uppercase | `<uppercase>text</uppercase>` | `<uppercase>text` |
| VoOffset | `<voffset=10>text</voffset>` | `<voffset=10>text` |
| Width | `<width=50%>text</width>` | `<width=50%>text` |
